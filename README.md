# Projet IWA 5ème année

Le projet est une application client/serveur avec un front JS et un back Java Spring Boot

# Lancement de la base de donnée Postgres

```bash
cd docker
docker-compose up
```

```
Utilisateur et base : postgres
Mot de passe : iwa
```
